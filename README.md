# Sistema de Busca PHP com Banco de Dados

Este projeto é uma implementação simples e eficiente de um sistema de busca (search engine) utilizando a linguagem PHP e integração com bases de dados (PostgreSQL). O objetivo é demonstrar como realizar consultas filtradas e exibir resultados de forma dinâmica.


## Funcionalidades

- Pesquisa de registos em tempo real ou via submissão de formulário.
- Conexão segura com a base de dados utilizando PDO ou MySQLi.
- Exibição de resultados formatados em HTML.
- Tratamento básico contra SQL Injection.

## Tecnologias Utilizadas

- **PHP**: Lógica do servidor e processamento de dados.
- **PostgreSQL**: Armazenamento de informações.

## Pré-requisitos

Antes de começar, vai precisar de ter instalado na sua máquina:
- Um servidor local (como **XAMPP**).
- PHP5.
- PGAdmin4
